# DevOps Tasks Board – Projet pédagogique CESI
Ce projet est une application web 3-tiers (Frontend React + Backend Node.js
+ Base PostgreSQL)
qui servira de support pédagogique pour :
- Docker & Docker Compose
- Kubernetes (Minikube)
- CI/CD
- Déploiement Render (Free Tier)
- Base PostgreSQL managée & scripts de restauration
## Structure du projet
devops-tasks-board
├── backend
├── frontend
└── infra
├── docker
├── k8s
└── db
## Objectif du TP
Créer, conteneuriser, déployer et documenter une application *cloud-native*
complète.
